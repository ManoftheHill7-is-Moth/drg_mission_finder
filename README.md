# drg\_mission\_finder

Prints the time that a specified mission will appear on the mission scanner in Deep Rock Galactic using the biome type, primary mission, season, etc.

Big thanks to rolfosian for creating https://doublexp.net/, the website that this finder pulls its data from.


Recommendations for maximizing xp chances and finding a "white whale" (50k+ xp):

Season 6 (ossium gems are worth a ton of xp, especially if the max of 7 are found)

Haz 5+ (2 2 2 2)

Azure Weald or Fungus Bogs (croppa spawns more frequently)

Mining Expedition 400 Morkite (morkite provides xp after primary quota is completed)

Not hollomite or dystrum (collect additional secondaries for a small amount of bonus xp)

Max Cave and Complexity (3)

Double XP and Lithophage Outbreak

Pots o' Gold (x4 gold mined. Provides more xp than Dark Morkite with x1.44 morkite mined)

Yuletide Event (error cubes have a 1:26 chance of spawning by breaking open presents)

A lot of luck!
